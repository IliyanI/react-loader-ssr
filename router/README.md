## Getting started
```
git clone https://github.com/IliyanI/react-loader-ssr.git <YOUR_FOLDER_NAME>
cd <YOUR_FOLDER_NAME>

```
### Install dependencies
```
 npm i
```
### Run the app in the development mode
Build and run the app on http://localhost:3001.
```
npm run dev
```
